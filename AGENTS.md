# Agent Guide

The full project guide lives in **[CLAUDE.md](./CLAUDE.md)**. Read it first for
orientation, architecture, conventions, and commands. This file exists so
AGENTS-aware tools can find the same instructions quickly.

Quick pointers:

- Deep context lives in `context/*.md`.
- Per-feature specs live in `context/features/`.
- Keep `context/current-feature.md` current before and after feature work.
- `npm run build` must pass before commit.
- Ask before committing.
- Commit messages carry no AI attribution.
