# Logiclings Brand Notes

## Name

**Logiclings**

Meaning:

- `Logic` = software reasoning, systems, patterns, foundations.
- `-lings` = small creatures, collectibles, playful learning companions.

Tagline:

> Tiny games. Big developer instincts.

## Logo Concept

A Logicling is a small rounded creature built from the visual language of software:

- Code brackets
- Nodes
- Connectors
- Loops
- Signals
- Tiny system paths

The included SVG is a starter mark, not a final polished brand package. Use it as a direction for favicon, app icon, mascot exploration, and early prototype identity.

## Alternate Names Considered

- Code Critters
- Devlings
- Stack Sprites
- Byte Buddies
- Patternlings
- Logic Garden
- System Sprouts

Logiclings is the strongest working name because it is cute, memorable, broad enough for AI/system design/agentic coding, and not tied only to syntax.
